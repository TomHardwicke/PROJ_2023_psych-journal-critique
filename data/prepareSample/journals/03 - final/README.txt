Instructions for obtaining data files

Files:
journals-random.csv
journals-prominent.csv

Instructions: 

For journals-random.csv, see the file "randomSampleJournals.R" for the code that created this file.
For journals-prominent.csv, this file is a direct copy of the file of the same name stored in the modified folder. This was manually created from the wos-jcr-psych-2022.csv file (in the '01 - from WOS' folder) which was downloaded directly from Web of Science Journal Citation Reports.
